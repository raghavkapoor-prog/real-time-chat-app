Welcome to ChatApp 😊

Real-time communication just got easier! ChatApp uses Angular for a powerful front-end, combined with Firebase services like Firestore and FireAuth to provide secure, real-time messaging.

Chat with anyone, anywhere, anytime! 🌍

What is ChatApp?

ChatApp is an alternative to existing messaging services like WhatsApp, offering real-time chat powered by Firebase Firestore. Experience seamless, fast, and reliable communication.

Benefits of Real-time chat:

  • 🚀 Instant updates and messages
  • 🔒 Secure communication using Firebase FireAuth
  • 💬 Scalable architecture with Firestore
  • 🎯 Angular Material for a sleek UI
